处理机调度的层次和调度算法目标

发表于2025-04-08|更新于2025-04-08|操作系统
|浏览量:

处理机调度：对处理机资源进行分配

高级调度 作业（外存）/长程调度

外存（作业）调到内存（进程） 运行频率低 将作业 -> 后备队列 批处理系统

低级调度 进程调度

就绪 -> 执行 获得处理机 抢占方式（优先权原则，短作业优先原则，时间片原则）非抢占方式（简单，开销小，难以满足紧急任务）
批处理系统 / 分时系统 / 实时系统

中级调度 中程调度（内存）

（暂时不能用的进程）内存中处于阻塞状态的 换到 外存挂起
内存 -> 外村（挂起） 外->内 中级

系统平衡（忘了）

CPU = cpu有效利用率 / （CPU有效利用率 + CPU空闲等待时间）
公平性 ： 时间片轮转
平衡性
策略强制执行

周转时间 = 提交 - 完成
平均周转时间
$$
\sum_{i=1}^n (提交-完成)
$$

带权周转时间 = 周转时间 / 服务时间
平均带权周转时间
$$
\sum_{i=1}^n (T/Ts)
$$
